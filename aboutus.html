How GearTrack Was Built

GearTrack was developed as part of the "Application Development for Analytics" course. The project requirements included implementing a structured, responsive website around a pre-assigned theme, sports equipment.

I began by translating the theme into a practical layout using HTML and CSS, focusing on clean design, category-based organization, and user-friendly navigation. I used a modular approach to construct individual pages and styled them with reusable CSS classes.

Throughout development, I applied responsive design principles, managed version control via GitHub, and deployed using Cyberduck. I incorporated instructor feedback and implemented enhancements like hover effects, subtle transitions, and interactive visuals to improve usability.

This project helped me strengthen my front-end development skills and demonstrate the ability to take a given theme and turn it into a polished, interactive website.

Technologies Used

  • HTML + CSS
  • JavaScript with Chart.js
  • GitHub for version control
  • Cyberduck for deployment
  • Visual Studio Code for development
